# Campus Student Management System

## Overview
This is a simple Student Management System built using Python and SQLite. It allows users to manage student records through a command-line interface.

## Features
- Add students
- View all students
- Search student by ID
- Update student details
- Delete student records

## Technologies Used
- Python
- SQLite

## Project Structure
```
main.py
student.py
database.py
```

## How to Run

```bash
python main.py
```
